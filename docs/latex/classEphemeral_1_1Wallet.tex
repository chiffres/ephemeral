\doxysubsection{Detailed Description}
\doxylink{classEphemeral_1_1Ephemeral}{Ephemeral} wallet whose balance decays over time. 

This wallet models a "{}melting"{} value system\+:
\begin{DoxyItemize}
\item Balance decreases continuously as time passes (entropy of money).
\item Any deposit or withdrawal first updates the balance to the current time, then applies the requested operation.
\end{DoxyItemize}

Conceptual purpose\+:
\begin{DoxyItemize}
\item Encourage circulation\+: value loses strength if it is left unused.
\item Embed the flow of time directly into the monetary object. 
\end{DoxyItemize}
